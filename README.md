# Uganda's Digital Mental Health Leap: Harnessing DHIS2 for Data-Driven Insights

This project implemented by Health Information Systems Programme (HISP), Uganda aims to leverage DHIS2, 
to digitize existing paper based mental health data from various referral hospitals and utilize machine learning and open-source analytics,
to generate actionable insights to inform mental health programming and support evidence-based policies

## Expected Outputs
- A harmonized nation-wide longitudinal mental health dataset capturing the longitudinal profiles of the individual patients
- A mental health data insights dashboard with key mental health indicators to support decision making
- Policy-relevant outputs, Open source research publications, and policy briefs

## Project Timeline
The project is expected to be completed within 12 months (May 2025 - April 2026)
